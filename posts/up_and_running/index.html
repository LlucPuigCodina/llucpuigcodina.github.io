Hi there! This is my first post for my personal webpage. Stay tuned to see more of them coming.

Let’s start with an incredible drawing from Akira. Why? Because Akira is awesome, that’s why. For those that might not know Akira, here they have a FAQ about the manga [SPOILERS], and I highly recommend reading it, aside from watching the animated movie.

As mentioned in the About section of my website, this website is based on the Jekyll/GitHub static site generator. Getting everything installed, setting up the repository and so on has been a pain in the ass since this is my first time using such tools. Specially the customization of the homepage to include the surrealist picture. The main tutorials I’ve used have been the following:

  • Jekyll Now
  • How to Create a Blog with Jekyll – A Beginner’s Guide
  • Build A Blog With Jekyll And GitHub Pages
  • Building a static website with Jekyll and GitHub Pages
  • Getting Started with Markdown
  • kmarkdown Syntax
  • BLOGGING WITH RMARKDOWN, KNITR, AND JEKYLL
  • How to add other themes for Jekyll Now?
  • Jekyll Series; best one for beginers I’ve seen.
  • R Markdown to Jekyll: “Protecting” Your Math Equations
