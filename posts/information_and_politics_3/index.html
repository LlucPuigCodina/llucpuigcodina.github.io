Smooth as consumption

Information and Politics III

January 6, 2017

The answers for the previous blog post.

  1. In the last period, tomorrow, there will be no more elections since the world ends. Go to the Career Concerns problem of the first post in this series and it is trivial to see that the optimal x for the politician to choose is 0 due to R = 0 (no value of re-election). When

\[ x = 0 \Longleftrightarrow r = b \]

and

\[ g = e^\eta x = 0 \]

Since g tomorrow is going to be 0, it doesn’t matter the value of \(\eta\) tomorrow, the politician’s ability/productivity/talent does not matter, the result will still be that no public goods are provided tomorrow. This is true for any politician that holds office in the last period. Due to this, the probability of being elected in the second election is not only the same for both politicians, \(\frac{1}{2}\) but ex-ante known. Because of this, the Career Concerns problem becomes (probability does no longer depend on what he spends today and some expectations, treated as given since the politician can’t determine what people think, on public good expenditure):

\[ \max_{x_t} \left \{ b - x_t + \frac{1}{2}*R \right \} \]

Since the probability is ex-ante given, he can’t favor his re-election chances by spending more on x today (everyone knows that in the second election they will elect a complete thief) and thus, sets x = 0, which implies that all of the budget today is allocated to rents: b = r. The model depends on some kind of Folk theorem.

  1. There must be a positive bias in the charisma utility of electing a challenger. This was defined as:

\[ \xi_i^C - \xi_i^I = \Psi_t + \psi_t^i + a \] where a = 0, \(\Psi_t\) follows a uniform distribution with mean 0 and density \(\Phi\) and \(\psi_t^i\) follows a uniform distribution of mean 0 and density \(\phi\). If we want to introduce a bias in favor of the challenger, motivated by his populist character, set $a ^{++} $. One will find that for large values of \(a\) the challenger always or almost always wins, thus politicians only hold one mandate and so the incumbent doesn’t assign budgetary resources to provide for public goods (x = 0 and r = b & g=0).

A voter i, will vote for the incumbent, only if:

\[ \Delta_i \geq \Psi_t + \psi_t^i + a \]

Public goods g and aggregate popularity \(\Psi_t\) are realized (we can take the aggregate popularity shock as-if realized since the individual shock is independent of preferences across voters). The share of members of group j who vote for the incumbent is:

\[ \begin{align} v_t^j(g_t, \Psi_t, a) &= \theta_j * Pr(\psi_t^i \leq \Delta_1 - \Psi_t - a) + (1-\theta_j)*Pr(\psi_t^i \leq \Delta_0 - \Phi_t - a)\\ &= \theta_j \left \{ \frac{1}{2} + \phi [ \Delta_1(g_t) - \Psi_t - a ] \right \} + (1-\theta_j) \left \{ \frac{1}{2} + \phi [\Delta_0 - \Psi_t - a]\right \}\\ &= \frac{1}{2} + \phi[\theta_j \Delta_1(g) + (1-\theta_j)\Delta_0 - \Psi_t - a] \end{align} \]

As a grows, specifically when it reaches the point where \(a = \frac{1}{2}\phi + \theta_j \Delta_1(g_t) + (1-\theta_j)\Delta_0 - \Psi_t\) the probability of the ith voter voting for the incumbent tends to 0:

\[ v_t^i (g_t, \Psi_t, a) \longrightarrow 0 \]

Thus, the probability of re-election when the populist effect is incredibly strong is:

\[ \pi(g_t) = Pr \left( \sum_{j=1}^J \lambda_j * 0 \geq \frac{1}{2} \right) = 0 \]

From the incumbents perspective:

\[ \pi(x_t) = \mathbb{E}[\pi(g_t) | x_t] = 0 \]

So, the Career Concern problem becomes:

\[ \max_{x_t} \left \{ b - x_t + R * 0 \right \} \]

Thus, the optimal investment in public goods is x = 0, so r = b. The share of rent extraction is given by:

\[ \rho \equiv \frac{r}{b} = 1 \]

  1. The incumbents Career Concerns problem is:

\[ \max_{x_t} = \left \{ b - x_t + R* \left( \frac{1}{2} + \bar{\theta} \Phi [ln(x_t) - ln(\bar{x})] \right) \right \} \]

Substituting in the bias (since now, the incumbent sets \(\bar{x}\) instead of taking it as given) and operating, we find that $ (x_t) = 1$. The optimization problem colapses to:

\[ \max_{x_t} = \left \{b - x_t + R*1 \right \} \]

Which yields an optimal x = 0 (x can’t be negative), and thus r = b. The net present value of rents under this bias is:

\[ NPV_{bias} = \delta (b + \delta *b + \delta^2*b + \delta^3*b + ... ) = \delta b \sum_{t = 0}^\infty \delta^t r = b*\frac{\delta}{1-\delta}; \ \ \delta \in (0,1) \]

In the standard model, the NPV of rents is the rent today r plus the expected value of re-election, R. In the model we find that:

\[ R = \frac{2\delta}{2-\delta} *r \]

\[ r = b \left(1 + \bar{\theta}\Phi\frac{2\delta}{2-\delta}\right)^{-1} \]

So:

\[ NPV_{standard} = b \left(1 + \bar{\theta}\Phi\frac{2\delta}{2-\delta}\right)^{-1} + \frac{2\delta}{2-\delta} b \left(1 + \bar{\theta}\Phi\frac{2\delta}{2-\delta}\right)^{-1} \]

It is always the case that \(NPV_{bias} > NPV_{standard}\). The rents the incumbent would be willing to forgo to obtain such bias are at most:

\[ NPV_{bias} - NPV_{standard} > 0 \]

  1. We found that rent extraction is a constant fraction of the budget:

\[ \rho(\theta) = \frac{r}{b} = \left( 1 + \theta \frac{2\delta}{2-\delta} \Phi \right)^{-1} \]

It is not difficult to see that this function is decreasing and strictly convex for $ (0,1) $. Then, let region l have a fraction \(\theta_l\) of informed voters and let there be any heterogeneity in $_l $ across regions. Then, by the previous properties described we know that by Jensen’s inequality we have that:

\[ \rho \left( \frac{1}{L} \sum_{l=1}^L \theta_l \right) < \frac{1}{L} \sum_{=1}^L \rho(\theta_l) \]

Which implies that when there is information heterogeneity across regions, a centralized government would extract a lower rent share than multiple, decentralized, governments.

The only problem with this is that there is some evidence of lower corruption in higher levels of government compared to local or federal ones and there is evidence of the informed voters mechanism, but there is no evidence or tests regarding this decreasing and convex properties of the function.

Obviously, this properties did not come up in an investigation, but where constructed by choosing appropriately the form of other aspects of the model (distributions, shocks, utility functions, public good production function and so on), so it all seems a bit ad-hoc, specially when no economic intuition is provided for this centralization effect.

In future posts I’ll try to provide an economic intuition about this, by adding wages to the politician, not only rents, and what happens when in a centralized government a politician receives the sum of wages of all of the politicians in the decentralization case.
